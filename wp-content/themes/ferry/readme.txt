ABOUT THEME
=========== 
ferry is Responsive, Multa-Purpose, HTML5 / CSS3 Theme for (Desktop, tablet, mobile phone…)
ferry is Created with Twitter Bootstrap 3.3.2 Framework. ferry is a great design idea for
website of Creative ferry,ecommerce, Corporate, Corporation and more … 
Create Outstanding Website or Blog in Minutes!. Awesome Design, Unique Concepts, ferry has static and fixed header feature. it’s developed with lots of care and love. 

Theme has two, three, four footer layout feature.We focused on usability across various
devices, starting with smart phones.it is compatible with various devices. ferry is a
Cross-Browser Compatible theme that works on All leading web browsers. ferry is easy to use and 
user friendly theme. 
Powerful but simple ferry theme content customized through customizer. to make your site attractive it has two 
widget sections first for “sidebar widget section” and second for “Footer widget section” . 

To make your website in two column use sidebar widget section. to set custom menu in header set primary location. we added social media links to added your social links.It boasts of beautifully designed page sections , Home, Blog and Default Page Template(page with right sidebar), Page Full-Width, Page Left Sidebar. ferry theme has more advanced feature to make your site awesome like: it has header top bar dark & lite color feature. Theme compatible with woocommerce. ferry is a fully woocommerce compitable theme. ferry is translation ready theme with WPML compatible & Many More….

This theme is compatible with Wordpress Version 4.5.2  and above and it supports the new theme customization API (https://codex.wordpress.org/Theme_Customization_API).

Supported browsers: Firefox, Opera, Chrome, Safari and IE10+ (Some css3 styles like shadows, rounder corners and 2D transform are not supported by IE8 and below).
